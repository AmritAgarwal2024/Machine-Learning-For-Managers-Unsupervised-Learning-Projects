# Machine-Learning-For-Managers-Unsupervised-Learning-Projects
This project includes 
  - Data Pre Processing which includes
      - Missing Data Imputation
      -  Numerical Encoding of Categorical Data
      -  Scaling of Non Categorical Data
  - Descriptive and Inferential Statistics
  - Graphical Representation Of Data
  - Clustering Of Records(Supervised Learning)
      - K - Means Clustering
      - Anglomerative Hierarchical Clustering
  - Comparing the Two methods for clustering and reprting the Findings and Managerial Insight.
